# 概要

艦これの編成一覧画像を作成する。



# 使い方

艦これ画面の以下のいずれかをキャプチャした画像を用意する

 * 編成 - 詳細 または
 * 改装

フォームにドラッグドロップする。

なお 7 個以上の画像がドラッグドロップされた場合 7 個目以降を無視する。
